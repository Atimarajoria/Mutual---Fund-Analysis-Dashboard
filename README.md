📌 Project Overview

The Mutual Fund Analysis Dashboard is a data analytics project designed to evaluate and visualize mutual fund performance using data-driven techniques. It helps users identify high-performing funds based on returns, risk, and other financial metrics.
This project combines Python, Excel, and Power BI to transform raw financial data into meaningful insights and interactive visualizations. Such dashboards are commonly used to analyze investment trends, risk, and returns for better decision-making.

🎯 Objective
Analyze large datasets of mutual funds
Identify top-performing funds with low risk
Compare different fund categories and investment strategies
Provide interactive insights using dashboards

🛠️ Tech Stack
Python (Pandas, NumPy, Matplotlib, Scikit-learn)
Excel (Data preprocessing)
Power BI (Dashboard & visualization)

📂 Project Structure

Returns
Expense Ratio
Fund Category
Risk Level
AUM (Assets Under Management)
📈 Key Insights
Equity funds generally show higher returns but higher risk
Expense ratio significantly impacts long-term gains
Diversified funds provide better stability
SIP investments help in risk averaging over time
